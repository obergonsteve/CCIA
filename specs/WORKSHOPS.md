LIVE WORKSHOPS

In this framework, there needs to be a calendar of scheduled 'live' workshops, webinars, Q&A sessions, tutorials, etc.

This will use LiveKit, as implemented in the GritHub project (for live meetings, chats, screen sharing etc.)

A 'workshop' is basically a special kind of 'unit', designed to be delivered 'live', with a sequence of content (video presentation, slideshow, test) like any other unit.

Workshop 'units' can be part of the content of other 'static' units - ie. steps in the process of completing a unit.  So the content/step will be like a 'link' to a 'live' workshop 'unit' on a given date, which will in itself involve the 'live' presentation of a video or slide deck, tests, etc.

In Admin, we need a section for setting up and planning these scheduled workshops.

In the User UI, there needs to be a 'Workshops' page to browse and 'join' (add to user's calendar).  This also needs to be possible from other 'static' units, which have 'live' units as part of the required content that must be completed along with the 'static' content.


3 tiers of Certification:

Bronze - mandatory - eg. required by law to operate within the industry.

Silver - optional - for users to build skills, specialise, etc.

Gold - optional - PD, leadership, conflict resolution, mental health, stress, etc.
