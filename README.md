# CCIA — Land lease training portal

CCIA Land Lease Division operator training (Next.js app).

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Configure Convex and env per `.env.example` (copy to `.env.local`).
